# APLICACION-ADMINISTRACION

Programa para la revision de archivos con plantilla de IMMS EMA y EBA, basado en las necesidades de la revisión bimestral de los archivos.
Unificando los datos en una sola hoja para su posterior trabajo y analisis


FORMAS DE LEER EL ARCHIVO

1.- INTERFAZ.PY     (ESTETICA DEL PROGRAMA)
2.- REQUISITOS.TXT  (LIBRERIAS PARA QUE FUNCIONE EL PROGRAMA)
3.- REVISION.PY     (CONTIENE TODA LA LOGICA DEL PROGRAMA)
4.- UTILIDADES.PY   (NORMALIZACION DE DATOS Y COLORES DE CELDA)
